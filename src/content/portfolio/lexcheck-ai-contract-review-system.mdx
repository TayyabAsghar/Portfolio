## Project Overview

At **LexCheck**, I worked on an AI-powered contract review platform that helps legal teams review, redline, and negotiate
contracts faster. The product integrates with **Microsoft Word**, applies preferred negotiation positions, highlights risky
language, and gives teams a structured way to manage contract review workflows.

My work covered both product delivery and engineering leadership. I contributed to the platform’s **frontend revamp and
rebranding**, built role-specific dashboard experiences, improved performance, strengthened security, and helped deliver internal
tooling around HubSpot CRM, Azure DevOps, and release automation.

## Product Context

Contract review is often slow, repetitive, and dependent on manual legal review. LexCheck reduces that friction by combining
contract lifecycle management with AI-assisted redlining, playbook-driven review, and secure document workflows.

The product needed to feel trustworthy and efficient for enterprise users. That meant improving dashboard clarity, protecting
sensitive contract data, supporting different user roles, and making the application faster for daily legal workflows.

## Key Contributions & Impact

I contributed as both an individual contributor and a lead, working across UI architecture, secure product workflows, API
integration, performance, testing, and delivery management.

### Product & Engineering

1. Worked as a core member of a **4-person frontend revamp team**, improving product usability, visual consistency, and sales enablement.
2. Improved the user experience and performance of the platform, contributing to a reported **20% increase in sales**.
3. Built **Role-Based Access Control (RBAC)** experiences with tailored dashboards for contracts, playbooks, users, and admins.
4. Improved application performance by **35%** through lazy loading, memory leak fixes, and removal of redundant UI dependencies.
5. Reduced bundle size by **10%** using Webpack Analyzer to identify duplicate code and unnecessary third-party packages.
6. Reached **100% frontend unit test coverage** by integrating Storybook into the component testing workflow.
7. Strengthened security with strict **CSP headers**, secure Web API usage, and workflows aligned with **SOC 2 Type II** expectations.
8. Supported infrastructure migration from **AWS to Azure DevOps**, helping reduce operational costs by **30%**.
9. Improved backend API reliability around onboarding and RBAC, increasing related test coverage by up to **50%**.

### Lead Role

1. Led a team of **3 developers** on HubSpot CRM implementation, including sales pipelines, workflow automation, invoices, and quotes.
2. Improved junior developer onboarding by documenting workflows and coding practices, increasing team efficiency by **40%**.
3. Managed sprint planning and backlog prioritization to keep feature delivery focused and predictable.
4. Created a single-file Sass theming system to speed up visual changes across Ant Design-based interfaces.
5. Set up linting standards, code review practices, and monthly coding sessions to improve implementation quality.
6. Configured **GitHub Actions** for deployment automation and monitored production behavior through **Azure Insights**.
7. Helped improve Agile delivery habits with Jira and Azure Boards, contributing to a reported **15% productivity lift**.

## Conclusion

LexCheck was a high-impact product experience across AI, LegalTech, performance, security, and leadership. It helped me grow from
shipping polished product features to leading delivery, improving engineering standards, and connecting technical decisions to
measurable business outcomes.
